About Weather App

Weather App is a simple web application that provides current weather conditions, astronomy data, and weather alerts for locations around the world.

Features

  • Current weather conditions (temperature, humidity, wind, etc.)
  • Astronomy data (sunrise, sunset, moonrise, moonset, moon phase)
  • Weather alerts and warnings
  • Search by city name, zip code, IP address, or coordinates

Data Source

This application uses the WeatherAPI.com service to retrieve weather data. WeatherAPI.com provides real-time weather information for locations worldwide.

Search Options

You can search for weather information using any of the following formats:

  • City name (e.g., "London", "New York")
  • City and country (e.g., "Paris, France")
  • US zip code (e.g., "90210")
  • UK postcode (e.g., "SW1")
  • Canada postal code (e.g., "G2J")
  • IP address (e.g., "100.0.0.1")
  • Latitude and longitude (e.g., "48.8567,2.3508")

Technologies Used

  • Python
  • Flask
  • Bootstrap 5
  • Font Awesome
  • WeatherAPI.com
